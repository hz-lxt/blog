一、用户交互Scanner

类似python的input的功能，只是java中做成了一个类

方法：

  • next() 监控用户输入的有效字符，有效字符之前的空白会自动去除，后续空白为结束符
  • hasNext() 是否存在输入
  • nextLine() 空白符不作为结束符，也不会去除前面的空白，已回车为结束符
  • nextInt()
  • hasNextInt() 同理
 public static void main(String[] args) {
               // 监控输入
        Scanner scanner = new Scanner(System.in);
               // hasNext来判断是否输入内容
        if (scanner.hasNext()){
              // next()方法会让用户等待输入
            System.out.println(scanner.next());
        }
        scanner.close();
    }
// 只输出了第一个空格前的数据

练习

输入数字就加起来，输入非数字就退出

public static void main(String[] args) {
    Scanner scanner = new Scanner(System.in);
    int i = 0;
    for (;;){
        if (scanner.hasNextInt()){
            i+=scanner.nextInt();
          //字符串.equals("被比较的字符串") 用于判断字符串是否相等
        }else if(scanner.next().equals("q")){
            scanner.close();
            break;
        }
    }
    System.out.println(i);
}

四、while循环

do while先执行一次再判断，while先判断再执行代码

 public static void main(String[] args) {
        int i = 1;
        int sum = 0;
        while (i<101){
            sum+=i;
            i++;
        }
        System.out.println(sum);
    }

do while

public static void main(String[] args) {
    int i = 101;
    int sum = 0;
    do{
        sum+=i;
        i++;
    } while (i<101);
    System.out.println(sum);
}

五、for循环

0-100奇偶总数

public static void main(String[] args) {
        int count1 = 0;
        int count2 = 0;
        for (int i =0;i<101;i++){
            if (i%2 == 1){
                count1+=i;
            }else {
                count2+=i;
            }
        }
        System.out.println(count1);
        System.out.println(count2);
    }

0-1000 5整除，三个一换行

public static void main(String[] args) {
    for(int i =1;i<1001;i++){
        if(i%5 == 0){
            System.out.print(i+"\t"); // print 不会默认输出换行符
        }
        if (i%15==0){
            System.out.println();
        }
    }
}

99乘法表

public static void main(String[] args) {
    for (int x=1;x<10;x++){
        for (int y=1;y<=x;y++){
            System.out.print(x+"*"+y+"="+x*y+"\t");
        }
        System.out.println();
    }
}
