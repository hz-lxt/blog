java 基础

引

  • 所有标识符都应该以字母，$，下划线开头
  • java是强类型语言，变量使用必须和服规定且先定义后使用

一、数据类型

1 基本类型（8个）

注意点：

  • 定义的时候不能超过范围
  • 定义long的最后需要加L，定义float的最后需要加F，使用的时候照常（不加问题不大）
  • 字符用char，单引号，字符串用String，双引号
  • String是类，不是基本数据类型

数值类型：

  • byte占一个字节，范围：-128-127
  • short占2个字节，范围：-32768-32767
  • int占4个字节，范围：-21亿-21亿
  • long占8个字节，范围-19位-19位
  • float占4个字节
  • double占8个字节

布尔类型：boolean，占一位（bit）

字符类型：char，占两个字节（byte）

整数扩展

进制

  • 二进制 0b开头
  • 十进制
  • 八进制 0开头
  • 十六进制 0x开头

3 类型转换

  • 在数据运算中，不同类型的数据先转换成相同类型的才能进行操作
  • 在转换的过程中需要注意内存溢出，即转换的类型不够容纳原来的数据长度
  • 强制类型转化：高容量数据类型-低容量数据类型
  • 自动类型转化：低容量数据类型-高容量数据类型
  • 转换时可能会存在精度丢失

从低到高

byte short char int long float double

强制类型转化格式
(类型)变量
byte b = (byte)122

内存溢出

public static void main(String[] args) {
        int i = 128;
              // byte内存溢出了，byte容量：-128-127
        byte b = (byte)i;
        System.out.println(i);
        System.out.println(b);
    }
128
-128

二、变量

  • java中变量是最基本的存储单元，其中包括变量名，类型，作用域
  • 类型可以是基本数据类型，也可以是引用类型
  • 实例变量是类的对象的变量，类变量是类本身的变量
  • 实例变量和类变量可以不初始化，这样使用就是默认值，局部变量必须初始化
  • 整数类型：0，浮点数：0.0 字符类型：’’，布尔类型：false，除了基本类型都是null
  • 除了第一个单词，后面单词首字母大写
public class Hello {
    static int a = 11; // 类变量
    int b = 22; // 实例变量
    public static void main(String[] args) {
        int c = 33; // 局部变量
    }
}

不同作用域变量的使用

public class Hello {
    static int a = 11; // 类变量
    int b = 22; // 实例变量 必须实例化类后才能使用
    public static void main(String[] args) {
        int c = 33; // 局部变量
        Hello hello = new Hello();
        System.out.println(hello.b);
        System.out.println(a);
        System.out.println(c);
    }
}
