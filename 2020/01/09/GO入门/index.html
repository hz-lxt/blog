一、语言介绍

1 简介

Go 即Golang，是Google公司2009年11月正式对外公开的一门编程语言。

Go是静态强类型语言，是区别于解析型语言的编译型语言(静态：类型固定 强类型：不同类型不允许直接运算)。

2 特性

跨平台的编译型语言

语法接近C语言

管道（channel），切片（slice），并发（routine）

有垃圾回收的机制

支持面向对象（但是go中没有类的概念）和面向过程的编程模式

3 版本

  • 2009年11月7日 weekly.2009-11-06 —— 早期的版本
  • 2018年8月24日 go1.11 —— modules和WebAssembly支持。
  • 2020 年 8 月 go 1.15

二、变量

注意点：

  • 变量定义了就必须使用，否则报错
  • 变量类型在定义阶段就决定了，无法改变
  • 变量推荐用驼峰体，文件名推荐用下划线分隔
// 变量的三种定义方式

// 1 全定义
// var 变量名 类型 = 值
//var a int = 123
//fmt.Println(a)

// 2 类型推导式
// 和上面的区别就是不用写类型
//var a = 123
//fmt.Println(a)

// 3 简略型
// 变量名 := 值
a := 123
fmt.Println(a)

// 4 声明多个变量
a,b:=123,456

// 5 按默认值定义
var age int // int类型默认值为0，string默认值为空字符串

// 坑
// 原来在go中，变量不能重复定义，但是在多个同时赋值时，左边只要有一个是未定义的，就可以这样使用
a:=123
a,b:=789,444
fmt.Println(a,b)

五、函数

// 有参，有返回值函数的调用与定义
func main() {
    msg,s :=add(5,6)
    fmt.Println(msg,s)
}
// 多个同类型参数可以省略类型，如果有返回值就必须定义时书写返回值类型，而且写了就必须一一对应返回
func add(a,b int)(string,int)  {
    return "加法",a+b
}

// 匿名函数
// 在go中函数是一等公民，可以把函数的内存地址传给变量
func main() {
    a := func() {
        fmt.Println("我是匿名函数")
    }
    // 要遵循变量被定义就必须使用的原则
    a()
}

// 函数的参数是函数，函数的返回值也是函数
func main() {
    add := func(a,b string) string {
        return a+b
    }
    newAdd := niuBiDeFunc(add)
    msg := newAdd("你好","帅")
    fmt.Println(msg)
}
func niuBiDeFunc(add func(a,b string) string) func(a,b string) string  {
    msg := add("我是","hz")
    fmt.Println(msg)
    return add
}

// 闭包函数 1 定义在函数内部 2 对外部作用域有引用
func test(age int) func()  {
    a:= func() {
        fmt.Println(age)
    }
    return a
}

六、变量作用域和范围

// 定义在全局的变量
var age = 18

func main() {
    fmt.Println(age)
    // 拿到的是全局的，改的也是全局的
    age = 19
    fmt.Println(age)
    setAge()
}

func setAge()  {
    fmt.Println(age)
    age = 20
    fmt.Println(age)
}

// 定义在全局和局部的变量
var age = 18

func main() {
    // 在main函数名称空间内用的是自己的age
    age := 0
    fmt.Println(age)
    age = 19
    fmt.Println(age)
    setAge()
}

func setAge()  {
    fmt.Println(age)
    age = 20
    fmt.Println(age)
}
