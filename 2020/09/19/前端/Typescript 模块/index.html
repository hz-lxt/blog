1 模块的定义

ts中内部模块现在称为“命名空间”，外部模块简称模块

ts中模块想要使用首先要在模块内部通过export把内的数据暴露出去，再在需要使用的地方import引入模块

2 模块的简单使用

被导入页面

export function printMsg(age:number,salary:number):number[]{
    return [age,salary]
}


var name = "hz"

var age = 18

var salary = 2000

// 默认导入，只能使用一次
export default salary

export{name , age}

导入页面

// 默认导入的方式就不需要加花括号，但是只能导入一个
import salary from "./modules/func"

import { printMsg as print,name,age } from "./modules/func"
// 可以多个一起导入，也可以起别名



console.log(print(18,2000))

3 命名空间的简单使用

命名空间就是在一个ts文件中，使用名称空间包裹一个范围，这个范围内的数据只属于这个名称空间，在外部可以重复

index.ts

// 如果其他页面要使用这个命名空间的数据就必须先导出namespace，再导出内部数据
export namespace A{
    export function printMsg(age:number,salary:number):number[]{
        return [age,salary]
    }
    var name = "hz"
    var age = 18
    var salary = 2000

} 
export namespace B{
    function printMsg(age:number,salary:number):number[]{
        return [age,salary]
    }
    var name = "hz"
    var age = 18
    var salary = 2000
}
